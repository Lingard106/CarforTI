# 两轮自平衡小车 - 2026 年重庆大学校级电子设计竞赛（B 题）

## 项目简介

本项目针对 **2026 年重庆大学校级电子设计竞赛 B 题（两轮自平衡小车）** 开发，基于组委会指定的 **STM32F407ZGT6** 主控板，实现两轮小车的自平衡控制、循迹行驶及定点停车功能。因开发时间限制，最终完成竞赛要求的核心功能模块，并提交完整设计报告。

## 系统设计

### 硬件架构

- **主控单元**：STM32F407ZGT6 开发板（组委会提供）
- **执行单元**：组委会提供的小车底盘（含直流电机、车轮、亚克力板）
- **传感单元**：
  - 姿态检测：陀螺仪/加速度计（MPU6050 等，用于自平衡解算）
  - 循迹检测：5 路 TCRT5000 红外传感器（GPIO 输入模式，软件消抖）
- **人机交互**：按键（启动）、LED（状态指示）、蜂鸣器（声光提示）
- **驱动单元**：电机驱动模块（支持 PWM 调速、电气刹车）

### 软件架构（FreeRTOS 多任务）

| 任务名称       | 优先级   | 核心功能                                                       |
| -------------- | -------- | -------------------------------------------------------------- |
| `BalanceTask`  | High     | 直立环（角度 PID）+ 速度环（编码器 PID）+ 转向环（循迹纠偏 PID） |
| `CarTask`      | Realtime | 按键检测、循迹状态机、声光提示、电机刹车逻辑                     |

### 核心算法

1. **自平衡控制**：基于角度 PID（PD 控制） + 陀螺仪阻尼，实现小车稳定直立。
2. **速度控制**：编码器测速 + 速度 PID 调节，保证匀速行驶。
3. **循迹控制**：5 路红外传感器加权偏差计算，通过差速转向纠偏。
4. **刹车逻辑**：行驶中软件平稳刹车（保持直立），静止后电气抱死刹车。

## 关键代码模块说明

- **`Balance_Task.cpp`**  
  直立环、速度环、转向环三环 PID 控制核心，实时读取陀螺仪角度与编码器速度，输出电机 PWM。

- **`Car_Task.cpp`**  
  整车逻辑任务：
  - 按键启动检测（一键启动）
  - 循迹传感器状态机（黑线检测、路径决策）
  - 停车逻辑触发（到达 B 点后先软刹再电气抱死）
  - LED + 蜂鸣器声光提示

- **`tcrt.cpp`**  
  红外循迹传感器驱动，采用 GPIO 输入模式 + 数字消抖采样，避免中断导致的误触发。

- **`bsp_dwt.h/c`**  
  基于 Cortex-M4 DWT 计数器的微秒级精准延时，兼容 FreeRTOS 系统。

- **`speed_pid.h`**  
  通用 PID 控制器封装（位置式/增量式），供速度环和转向环复用。

## 测试结果

### 稳定站立测试

- 启动方式：一键按键触发  
- 稳定时长：≥3 秒  
- 位移范围：≤10 cm（优于竞赛要求的 15 cm）  
- 测试环境：平整白色哑光地面

### A→B 点行驶测试

- 起点：A 点（半圆弧顶点）  
- 终点：B 点（半圆弧顶点）  
- 触发条件：5 路红外传感器检测到黑线（线宽 1.8 cm）  
- 停车逻辑：首先软件平稳刹车（保持直立），随后电气抱死刹车  
- 声光提示：LED 常亮 + 蜂鸣器鸣响 500 ms  
- 重复测试：10 次测试均精准停在 B 点，无误触发 / 漏触发

## 未完成模块说明

因校赛开发周期预留不足，以下功能留待后续完善：

## 快速开始

### 环境依赖

- 开发工具：STM32CubeMX + Clion  
- 固件库：STM32F4xx HAL 库  
- 操作系统：FreeRTOS（兼容 DWT 硬件计时）  
- 硬件烧录：ST-Link

### 编译与烧录

1. 用 Keil MDK5 打开工程文件。
2. 选择芯片 `STM32F407ZGT6`，编译（Build）。
3. 连接 ST-Link 调试器，点击 Download 烧录至开发板。

### 运行步骤

1. 将小车放置在 A 点（起跑线位置）。
2. 按下启动按键（已配置 GPIO 中断或轮询）。
3. 小车自动直立 → 沿黑线行驶至 B 点 → 检测到黑线触发停车 → LED 常亮 + 蜂鸣器鸣响。

## 总结与改进

### 完成亮点

- 严格遵循竞赛规则（仅使用指定主控、无摄像头）。
- 自平衡稳定性优异，位移控制优于竞赛要求（≤10 cm）。
- 循迹传感器采用 GPIO 输入 + 数字消抖，无中断抖动误触发。
- 刹车逻辑适配平衡车特性：行驶中平稳刹车，静止后电气抱死。

### 改进方向

- 补充多定点位置标定算法，实现 A/B/C/D 四点精准识别。
- 优化弧线转向 PID 参数，适配 40 cm 半径半圆弧行驶。
- 增加圈数计数与路径切换逻辑，完成多圈连续行驶。
- 基于 DWT 计时优化速度控制，压缩绕圈耗时。
